# PoiPoi - Eidi Expense & Tracker 

**PoiPoi** is a fun, lightweight, and highly interactive web application designed to keep track of your Eid salami/Eidi. It helps you manage your list of loved ones, log who gave you money, filter by Eid festivals/years, and instantly track who is yet to contribute! The app is fully responsive and optimized for both mobile and desktop screens.

---

##  Features

* **  Dashboard Overview:** Get a quick glance at the total number of enlisted loved ones and the grand total of collected Eidi.
* ** Loved Ones (Master List):** Add your family and friends to a centralized list and track the total amount received from each individual.
* ** Smart Data Entry:** Easily log new Eidi entries by selecting the person, target year, and specific festival (Eid-ul-Fitr or Eid-ul-Adha).
* ** Contributors &  Defaulters:** Filter by year and Eid type to instantly see who already gave salami and who is yet to pay!
* ** Custom Themes & Audio:** Includes Dark Mode toggle and ambient background music tailored for each section to elevate the user experience.
* ** LocalStorage Integration:** No database configuration needed! All data is securely saved in your browser's `localStorage`, ensuring your logs persist even after a page refresh.

---

##  Tech Stack

* **HTML5** - Structured markup and semantic layout.
* **CSS3** - Custom styling, CSS variables for dark/light themes, and mobile-responsive breakpoints.
* **JavaScript (Vanilla)** - Dynamic filtering, drop-down toggle interactions, audio playbacks, and LocalStorage logic.

---

##  How to Run the Project

1. Open the `index.html` file directly in any modern web browser (No server or setup required!).
